# camList_task

## Challenge 1
### install on linux/unix like systems
```
clone the repo
cd camList_task/petstore_project
python3 -m venv vnev
pip install -r requirments.txt
source venv/bin/activate
cd petstore_project/
python manage.py runserver
```
### Run tests
```
python manage.py tests --verbosity=2
```
